## 🚀 使用说明

### 快速开始
1. **扫码访问**: 使用手机扫描上方二维码
2. **浏览选择**: 查看数据集清单,选择需要的数据集
3. **获取数据**: 根据权限查看或下载对应数据示例
4. **协作反馈**: 发现问题或有改进建议,欢迎反馈

### 数据权限说明
- **公开数据集**: 所有人可查看,支持复制和导出
- **受限数据集**: 需申请权限或联系管理员获取访问资格
- **付费数据集**: 需购买或订阅后才能访问完整内容

### 引用规范
使用本导航站的数据集时,请注明:
```
数据来源: 成品数据集导航站 (GitHub)
数据提供者: [具体数据集提供方]
访问日期: YYYY-MM-DD
```
